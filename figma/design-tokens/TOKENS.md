# design-tokens

Design tokens generated by [Tokener](https://www.figma.com/community/plugin/tokener).

## Installation

```bash
npm install design-tokens
```

## Usage

**JSON:**

```javascript
import tokens from 'design-tokens'
```

**CSS:**

```css
@import 'design-tokens/css';
```

**Tailwind:**

```javascript
// tailwind.config.js
const tokenTheme = require('design-tokens/tailwind')

module.exports = {
  // ... your config
  theme: {
    extend: {
      ...tokenTheme.theme.extend,
    },
  },
}
```

> **Note:** Tailwind config uses CSS variables for theme support. Import the CSS file alongside for mode switching.

## Updating Tokens

1. Open the Figma file and run the **Tokener** plugin
2. Click **Export Package** and download the zip
3. Extract the `dist/` folder into this repository, replacing existing files
4. Commit and push to a new branch:
   ```bash
   git checkout -b token-update/$(date +%Y%m%d)
   git add dist/
   git commit -m "chore: update design tokens"
   git push -u origin HEAD
   ```
5. Create a Pull Request for review
6. Once approved and merged, CI will publish the new version to npm
